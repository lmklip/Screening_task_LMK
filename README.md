# Screening_task_LMK
Отборочное задание для участников конкурса
## Как выполнить задание
1. Сделай fork этого репозитория.
2. Клонируй свой форк:
   git clone https://github.com/<your_Full_name>/<repo>.git
3. Выполни задание в папке.
4. Закоммить:
   git add .
   git commit -m "ФИО"
5. Отправь в свой репозиторий:
   git push
6. Создай Pull Request в основной репозиторий.
