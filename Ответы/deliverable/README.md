# Основная информация

В папке 2 скрипта, import_data.py и main.py. Для запуска main.py необходимо устонавить PyQt5 последней версии.


- import_data.py содержит создание базы данных и наполнение её из предоставленных файлов csv, логирование ошибок при возникновении.
- main.py содержит графическую и техническую часть приложения.

**Что было созданно?**


Графическое приложение, с реализацией функционала заказа товаров (Выбор пользователя, выбор товара, его количества, формирование корзины с информацией),
базовой продуманной логикой (пример, нельзя заказать товар при отсуствии его на складе) и полноценной записью заказа в БД (см код). Сам код закоментирован и удобно структурирован для чтения.


**Что не было созданно?**


К сожалению я не создал API и все вещи связанные с ней из-за нехватки знаний. Так же не понял сценарии использования.



**Реализация import_data.py**


В начале кода идёт создание структуры БД. После, функции чтения каждого csv файла и запись в БД. В функции main (которая вызывается при запуске файла) по 
очереди подставляются каждые функции чтения и записи, ошибки при возникновении логируются

**Реализация main.py**


Весь скрипт закомментирован, намного удобнее будет прочесть именно его. Реализован графический интерфейс библиотекой PyQt5
